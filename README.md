# eViewer Document Compare SDK Sample

The **eViewer Document Compare SDK Sample** demonstrates how developers can leverage the eViewer framework to compare two documents side-by-side with precision and efficiency. This sample allows users to visually analyze differences, highlight changes, and navigate through variations with ease. It provides a lightweight yet powerful solution for integrating robust comparison features into a web application.

Whether you are validating revisions, reviewing updated content, or
ensuring document accuracy, this sample illustrates how the eViewer
Document Compare SDK simplifies the comparison workflow while delivering
a smooth and intuitive user experience.

## Set up eViewer v7

Follow the steps below to run the eViewer Document Compare SDK Sample:

1.  Clone this repository and host it via an HTTP server.
2.  Download the viewer static files from the **eViewer7 npm repo** into
    the `viewer` folder.
3.  Ensure all viewer files are placed inside the root `viewer`
    directory.
4.  Open **eViewerPage.html** in your browser to launch the viewer.

## How It Works

Follow the steps below to explore and use this feature effectively:

1.  Open **eViewer** in your browser to launch the comparison
    interface.
2.  Choose the **Compare** option from the top toolbar.
3.  Select the two documents you want to compare from the available
    options.
4.  Start the comparison process to let eViewer analyze both documents.
5.  The viewer highlights all detected differences and displays them in
    an easy-to-navigate results panel.
6.  Utilize the navigation and **Filter** controls to navigate between differences or refine comparison results based on the type of differences, such as textual, graphics, or annotations, and specify which changes you wish to view.
7.  Enable **synchronized scrolling** to review both documents side by
    side with matched zoom and alignment.
8.  Exit compare mode anytime to return to the standard viewing
    interface.
